# Інструкція зі збирання та запуску проєкту

## Вимоги до середовища

Для коректної роботи проєкту необхідно:
- Операційна система: Windows 10 / 11
- CLion (2023 або новіша)
- PyCharm (2023 або новіша)
- CMake >= 3.20
- Компилятор C++: MSVC або MinGW з підтримкою C++17
- Python >= 3.13
- Мережевий стек Winsock2 (входить до складу Windows)

## Відкриття та збірка C++ серверної частини в CLion
1. Запустити CLion
2. Обрати File -> Open та відкрити кореневу директорію проєкту
3. CLion автоматично зчитає файл CMakeLists.txt і налаштує середовище збірки
4. Переконатися, що у CMakeLists.txt задано стандарт:
`set(CMAKE_CXX_STANDARD 17)`

5. Дочекатися завершення індексації та генерації конфігурації

Збірка виконується автоматично при запуску.

## Запуск сервера в CLion
1. У правому верхньому куті CLion обрати конфігурацію запуску
2. Натиснути кнопку Run

Під час запуску сервер:
  - автоматично обирає файли відповідно до варіанту
  - паралельно будує інвертований індекс
  - виводить статистику побудови індексу
  - починає прослуховувати TCP-порт 8080

> Сервер не приймає аргументи командного рядка - усі параметри (порт, кількість потоків, шлях до даних) зафіксовані в main.cpp

## Запуск C++ клієнта в CLion
1. Відкрити директорію cpp_client у CLion
2. Запустити клієнт після запуску сервера

Після підключення клієнт переходить у інтерактивний режим.

## Запуск Python-клієнта в PyCharm
1. Відкрити директорію python_client у PyCharm
2. Перевірити інтерпретатор у правому нижньому кутку
3. Відкрити файл main.py
4. Запустити програму (Run)

## Доступні команди клієнтів
- search \<term> - пошук документів за терміном
- search_and \<t1> \<t2>... - пошук документів з усіма термінами
- search_or \<t1> \<t2> ... - пошук документів з будь-яким терміном
- stats - статистика індексу
- freq \<term> - частота терміна
- add_file \<server_path> - додавання нового файлу до індексу
- help - вивести список команд
- exit - завершення роботи клієнта

## Завершення роботи
- Сервер: кнопка Stop у CLion або Ctrl + C
- Клієнти: команда exit або завершення програми

Сервер коректно закриває сокети та звільняє ресурси.
